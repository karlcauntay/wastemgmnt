Main Project Objective

The main objective is to create an IoT-based Automated Drainage Clog Detection and Debris Management System powered by solar energy that helps to reduce flooding in the city. The system automatically detects and removes debris in drainage canals to keep water flowing smoothly without the need for people to clean it often. This project aims to make drainage maintenance smarter, more efficient, and eco-friendly for future smart cities.

Project Components Overview

ESP8266

The core Wi-Fi enabled microcontroller, responsible for data collection, processing, and communication.

ESP8266 Exapansion board

Simplifies power distribution and component interfacing, making prototyping easier and safer.

Obstacle IR Sensor

Detects the presence of solid objects or blockages (debris) within the drainage pipe.

Sevo Motor MG996R

A high-torque motor used to control the physical gate mechanism for debris management.

Moduled LED RGB

Provides quick, color-coded visual feedback (e.g., green, yellow, red) on the system's status.

LCD 16x2 Built in I2C

Displays real-time sensor readings and text-based warnings locally.

JumperWIre m-f f-f

Essential flexible conductors for creating temporary connections between the components.

2inch Water Pipe

Forms the physical model of the drainage system where detection occurs.

Bucket

Serves as a reservoir or collection point for testing water flow and debris capture.

2 storage box

Containers for safe and organized housing of electronic parts and mechanisms.

cardboard

Inexpensive material used for initial mounting and rapid construction of the prototype frame.

Required Libraries

These are the necessary Arduino/C++ libraries for compiling and running the project code on the ESP8266 microcontroller.

Library File Purpose in Project
ESP8266WiFi.h Manages the Wi-Fi connection of the ESP8266 module to the network for Internet of Things (IoT) functionality.
ESP8266WebServer.h Allows the ESP8266 to host a web server, creating a local dashboard or API endpoint for monitoring and control.
Servo.h Provides functions for controlling the position of the Servo Motor (MG996R) to operate the debris management gate.
Wire.h Enables communication using the I2C protocol, which is necessary for interfacing with the LCD 16x2 display module.
LiquidCrystal_I2C.h Specific library for easily writing and controlling text displayed on the I2C-enabled 16x2 LCD screen.
